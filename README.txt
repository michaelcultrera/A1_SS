To compile and run the code using the COMMAND LINE:

- navigate to appropriate directory
- paste 1st command: export PATH_TO_FX=/Users/Michael/Documents/javafx-sdk-13.0.2/lib
- paste 2nd command: javac --module-path $PATH_TO_FX --add-modules javafx.controls,javafx.fxml filename.java 
- paste 3rd command: java --module-path $PATH_TO_FX --add-modules javafx.controls filename (without the .java extension)
